KOCO — 코딩 역량 성장 지원 서비스

매일 도착하는 해설과 문제 추천으로 만드는 코테 루틴

서비스 개요

KOCO는 생성형 AI를 활용하여 코딩 테스트 역량 향상에 도움을 제공하는 서비스입니다. 오늘의 문제를 풀고 간단한 설문을 남겨 문제 해설집을 제공받고, 모르는 문제는 챗봇과 커뮤니티를 활용해 해결 가능합니다. 또한 사용자의 학습 현황을 추적하고 개인화된 문제 추천을 제공하여 효율적인 학습 환경을 제공합니다.

빠르게 생각하고, 똑똑하게 행동하는 인재를 양성하는 것이 저희 서비스의 목표입니다.

  • 코테 습관 형성: 매일 도착하는 해설과 문제 추천으로 루틴 구축
  • 학습 효율 증대: 챗봇을 통한 문제 해결과 라이브 코딩 테스트 면접 연습
  • 다각적 몰입: 코드 공유 커뮤니티와 미니게임으로 동기 강화

주요 기능

해설: 문제 데이터를 기반으로 AI가 문제 요약, 풀이 전략, 최적의 풀이를 생성합니다.

챗봇: 이전 대화 내역과 문맥 요약문을 사용하여 대화를 이어갑니다. 각 역할별 에이전트가 상호작용하여 응답을 생성하고 스트리밍합니다.

문제 추천: 사용자의 설문 데이터를 기반으로 문제를 추천합니다. 알고리즘과 문제 설명에 대해 각각 임베딩 유사도를 비교합니다.

서브 기능

파이프라인

해설 제공 파이프라인 Explainer

  1. 스케줄러로 문제 크롤링
  2. Pydantic으로 스키마 검증
  3. 알고리즘 공식 문서 RAG 컨텍스트로 주입
  4. FAISS 리트리버로 관련 문서 선별
  5. Gemini API를 통해 구조화된 JSON 생성
  6. 백엔드 API로 전송

문제 추천 파이프라인 Recommender

  1. 입력: 전날 문제(2개) + 사용자 설문
  2. 난이도와 알고리즘 태그로 1차 후보 필터
  3. 앙상블 리트리버: BM25(태그/설명), FAISS(의미)
  4. 가중치 적용: 0.5 / 0.1 / 0.4
  5. 최종 랭킹: 추천 세트 4개 생성
