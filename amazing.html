AMAZING — 생성형 AI 기반 미로 게임 서비스

나만의 세계관, 픽셀로 그리는 모험

↓

프로젝트 개요

Amazing은 사용자가 원하는 대로 스토리를 생성해 매번 새로운 경험을 제공하는 생성형 AI 기반 미로탈출 게임 서비스입니다. 사용자 입력을 기반으로 전체 스토리, NPC 대사, 퀴즈, 배경 이미지까지 동적으로 생성되며, 플레이어가 직접 세계관을 만들어가는 독창적인 인터랙티브 경험을 제공합니다.

궁극적으로는 사용자가 매번 스스로 이야기를 만들어가는 즐거움을 제공하는 게임 경험을 목표로 합니다.

  • 실시간 스토리 생성: 세계관과 분위기 기반으로 시나리오를 전개합니다.
  • NPC 상호작용: 캐릭터 대사·퀴즈·행동을 동적으로 구성합니다
  • 픽셀 이미지 생성: 장면과 배경 이미지를 동적으로 생성합니다
  • 매번 새로운 경험 제공: 플레이마다 다른 흐름,주제로 플레이 가능합니다
기획 의도

“사용자가 직접 세계관을 만드는 AI 게임”이라는 아이디어에서 출발했습니다. 입력에 따라 매번 다른 스토리, NPC, 대사, 배경 이미지를 통해 몰입형 인터랙티브 게임 경험을 제공하고자 했습니다.

주요 기능

스토리 생성: 사용자 입력(세계관/분위기/키워드/난이도)을 기반으로 전체 세계관을 생성하고, 진행 상태에 맞춰 후속 전개를 동적으로 이어갑니다.

NPC·퀴즈: NPC 캐릭터별 역할/성향을 반영해 대사와 퀴즈를 구성합니다. 정답,오답 분기 및 힌트 제공 로직을 포함합니다.

이미지 생성: 픽셀아트 스타일 배경/오브젝트/캐릭터 이미지를 실시간 생성하여 몰입감을 높입니다.

파이프라인

스토리 생성 파이프라인

  1. 입력 수집: 세계관/분위기/난이도/키워드
  2. LangChain 체인 구성: 프롬프트 템플릿 + 메모리/컨텍스트
  3. Pydantic + with_structured_output 으로 JSON 스키마 검증
  4. 스토리·NPC·퀴즈(정답,오답,힌트) 구조화 생성
  5. 예외 처리,재시도 로직 처리
  6. JSON 스트리밍,배치 응답 반환

이미지 생성 파이프라인

  1. 입력: npc 대사 장면 요약 + 스타일(픽셀 아트,레트로 감성) + 이미지 해상도
  2. 프롬프트 구성: 프롬프트(로우 폴리,픽셀화,도트 감도) 주입
  3. DALL·E 3 API 호출을 통해 이미지 URL 생성
  4. 캐싱,리사이징,폴백 처리
  5. 클라이언트로 이미지 전달 및 스토리 전환

전체 서비스 아키텍처

담당 역할 및 기여

  • FastAPI 기반 RESTful API 서버 설계·구현
    엔드포인트/스키마/비동기 처리 설계 및 구현
  • LangChain 기반 LLM 응답 구성
    세계관/분위기 입력 시 스토리/퀘스트 흐름, NPC 대사/퀴즈 자동 생성 구현
  • 픽셀 스타일 이미지 생성 파이프라인
    DALL·E 3 API를 통한 이미지 생성, 프롬프트 가이드 및 후처리 구현
  • 구조화 출력/예외 처리
    Pydantic + with_structured_output 으로 스키마 검증, 예외 처리 로직 구현
  • 아키텍처 간소화
    초기 Spring 중계 서버 제거 후 AI 서버 단독 운영 구조로 통합
